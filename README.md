# NO-MissionMenuSearch

A mod to improve the mission menu with a search bar! 

It's not super complicated. It just adds a search bar that searches the description and the titles for missions to find matches and auto percolates them into the list. Less time spent scrolling, more typing. Its a purely QOL change that I made for myself as I find it annoying to scroll down a list to find a mission I want to play when I know the title for it. 

Though, I do want to use this to springboard off and do some menu improvements. Some things for the future I want to try: 

- Improved weapon selection view
    - Add more stats that are visible to the weapon to help with some hidden stats and mechanics like how the ATPs wont go over terrain, or the ARADs will pivot off mark to make it harder for AD to get all of them when saturating a target.
    - Possibly go the ace combat route of also adding small gifs animating how it slams into a target.

If there are any issues, come find me on discord (@spinypine) with your logoutput.log file.
